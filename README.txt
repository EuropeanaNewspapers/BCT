BCT - Binarization and Conversion Tool
------------------------------------------

This tool is a simple command line wrapper written in C# to create binarized and compact viewing files from large document images files.

This tool was written for the European Newspaper project (http://www.europeana-newspapers.eu/).

The following libraries are required:
	- GraphicMagick (http://www.graphicsmagick.org/) - gm.exe must be in the PATH
	- Kakadu compress (http://kakadusoftware.com/) - kde_compress.exe must be in the PATH
	- A binarization tool developed by Basilis Gatos et al. at NCSR (http://users.iit.demokritos.gr/~bgat/) - Binarization.exe must be in the PATH
		Note that this tool is *not* freely available. Please contact Basilis Gatos (bgat@iit.demokritos.gr) if you want to use the binarization!
